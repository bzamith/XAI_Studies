1. Introduo

Voltar Tabela de Contedos

Este notebook visa resolver a atividade proposta, com o seguinte enunciado:

Treinem um modelo caixa-preta (e.g., Random Forest ou outro modelo pra variar) para uma base de dados do seu interesse. Crie um global surrogate caixa-branca (e.g., rvore de Deciso), que simule esse modelo caixa-preta aprendido. Faam o upload do notebook.

Observe em que situaes o Global Surrogate apresenta uma fidelidade alta em relao ao modelo caixa-preta.

3. Conjunto de Dados

Voltar Tabela de Contedos

A conjunto de dados escolhido foi o "Predict Students' Dropout and Academic Success", do UCI (link).

Ele advm do paper "Early prediction of student's performance in higher education: a case study", de Mnica V. Martins, Daniel Tolledo, Jorge Machado, Lus M. T. Baptista, e Valentim Realinho. 2021. Publicado em "Trends and Applications in Information Systems and Technologies".

A descrio dele dada como:

Um conjunto de dados criado a partir de uma instituio de ensino superior (adquirido de vrios bancos de dados distintos) relacionado a estudantes matriculados em diferentes cursos de graduao, como agronomia, design, educao, enfermagem, jornalismo, gesto, servio social e tecnologias. O conjunto de dados inclui informaes conhecidas no momento da matrcula dos estudantes (caminho acadmico, dados demogrficos e fatores socioeconmicos) e o desempenho acadmico dos estudantes ao final do primeiro e do segundo semestres. Os dados so usados para construir modelos de classificao para prever a evaso e o sucesso acadmico dos estudantes. O problema formulado como uma tarefa de classificao em trs categorias, na qual h um forte desequilbrio em direo a uma das classes.

A descrio de cada uma das features pode ser encontrada na pgina do UCI.

Temos 3 classes distintas nesse caso: Graduate, Dropout e Enrolled.

No entanto, optei por excluir os alunos cujo status "Enrolled", afinal eles ainda esto passveis de evadirem ("Dropout").

Portanto, com base na suposio acima, posso tratar o target como binrio, removendo todos os exemplos de "Enrolled". Por fim, converti a coluna "Target": "Graduate" para 0 (negativo) e "Dropout" para 1 (positivo).

In[8]:
df = df[df.Target.isin(['Graduate', 'Dropout'])].reset_index(drop=True)
df['Target'] = df.Target.map({'Graduate': 0, 'Dropout': 1})

5. Modelo Global Surrogate

Voltar Tabela de Contedos

In[17]:
models['surrogate_1'] = DecisionTreeClassifier(
    criterion='entropy',
    max_depth=50,
    min_samples_leaf=5,
    random_state=42
).fit(X_test_scaled, y_preds_test['black_box'])
In[18]:
y_preds_test['surrogate_1'] = models['surrogate_1'].predict(X_test_scaled)
In[19]:
r2 = r2_score(y_preds_test['black_box'], y_preds_test['surrogate_1'])
round(r2, 2)
Out[19]:
0.91
O modelo Global Surrogate obteve uma pontuao r2 igual a 0.91, o que tambm considerei satisfatrio. Ou seja, um bom indicativo de que o modelo Global Surrogate aprendeu a replicar bem o modelo Black-Box.

Por fim, preciso atentar ao fato de que no captulo 8.6 do livro Interpretable Machine Learning - Christoph Mohnar, o autor destaca que no existe um cut-off exato para o r2.

Como vimos, o modelo Global Surrogate parece bem fiel ao modelo Black-Box. No entanto, ainda dificil interpret-lo pois ele est relativamente complexo.

Vamos tentar deixar o modelo Global Surrogate mais simples, para maior interpretabilidade.

In[21]:

Agora o modelo ficou muito mais fcil de ser interpretado, inclusive destacando as features mais importantes.

Contudo, com a pontuao r2 igual a 0.68, perdeu-se muito da fidelidade do modelo Global Surrogate em relao ao modelo Black-Box.

Assim, importante sempre fazer o trade-off e otimizar a Fidelidade com Interpretalidade.
